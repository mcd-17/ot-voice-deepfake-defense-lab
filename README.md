# 🏭 OT Voice Deepfake Defense Lab  
### AI-Driven Social Engineering Defense for Industrial Control Systems  
**Author:** Darshana M Chigari  

---

## 🚀 Overview

The OT Voice Deepfake Defense Lab is an advanced industrial cybersecurity research prototype designed to simulate and defend against AI-driven voice social engineering attacks targeting industrial operators.

This system models a real-world scenario where an attacker impersonates a Chief Engineer using deepfake voice technology to request critical safety overrides inside an industrial control environment.

The platform integrates:

- 🎙 Voice Fingerprinting & Behavioral Biometrics  
- 🧠 5-Layer AI Detection Engine  
- 🏭 Virtual PLC / OT Environment  
- 🔐 Secure Command Confirmation Protocol  
- 📊 Real-time Threat Scoring & Logging  

This is 2026+ threat modeling for Operational Technology.

---

## 🎯 Problem Statement

Industrial plants rely heavily on verbal command approvals for:

- Safety interlock bypass
- Emergency shutdown overrides
- Pressure alarm suppression
- Maintenance authorization

With the rise of AI voice cloning, attackers can now impersonate plant executives or engineers with high accuracy.

This system prevents that.

---

## 🧠 5-Layer AI Detection Architecture

### Layer 1 – Spectral Analysis
- FFT
- Pitch deviation detection
- MFCC comparison

### Layer 2 – Formant Fingerprinting
- F1/F2 resonance matching
- Vocal tract signature validation

### Layer 3 – Neural Artifact Detection
- GAN / TTS artifact scanning
- Phase inconsistency detection
- Jitter & shimmer variance analysis

### Layer 4 – Prosody Behavioral Modeling
- Speech cadence profiling
- Pause distribution analysis
- Authority tone patterns

### Layer 5 – Context Authority Validation
- Role-based access control
- Maintenance window verification
- Command risk scoring
- Dual-approval enforcement

Even if voice matches — unauthorized commands are blocked.

The simulator models:

AI intercepts before execution.
